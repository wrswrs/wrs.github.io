初识 Hexo

为什么是Hexo + Github

常见的静态博客框架有 Hexo、Hugo、Jekyll 等等，这里博主使用的是 Hexo。为什么使用 Hexo 和 Github Pages？

  • Hexo 拥有庞大的社区 / 开源支持，而且博主作为一名前端程序员，基于 Node.js 的架构使用起来亲切。
  • Hexo 提供的主题种类繁多，有些很好看，同时有着丰富的插件。
  • 使用Github这样免费的空间即可以节省一些费用，也节省一些打理的时间，只需要专心写文章。

环境准备
